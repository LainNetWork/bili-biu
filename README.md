### BILI-BIU B站收藏夹视频定时备份工具
一个部署在服务器上的监控BiliBili收藏夹并自动下载备份视频的工具，自带一个webUI
#### 预计功能：

- 定时监控指定收藏夹，并自动下载保存视频（已完成）
- WEB UI(简陋版)
- WEB UI 登录验证(目前还是初版，暂时没加)
- 单视频多线程下载（开发中
- 命令行模式（未开发
- 弹幕备份（未开发
- 定时备份上传谷歌云盘（遥远的未来
- 通用化任务，添加P站定时备份收藏夹模块（地球毁灭之时
